Открыть в Python бинарный файл в BMP­формате. Вычитать из BMP­заголовков
файла размер всего файла и общий размер заголовков BMP. Заменить все пиксели в
BMP­файле на пиксели черного(либо белого цвета). Записать полученную картинку в
файл на диске. Имена входного и выходного файла передать через параметры
командной строки, доступ к ним в Python можно получить через список sys.argv.
